# UtilizandoTADemC-
Código estudado e desenvolvido na matéria de estrutura de dados(UTFPR - Universidade Tecnológica Federal do Paraná), utilizando linguagem em C, Tipos abstratos de dados e ponteiros.

  Código projetado como simulação de um "Banco" de forma simples, contendo o básico como criar conta, depositar, sacar, transferir, extrato e sair da aplicação, utilizando o método tad para organizar e facilitar o desenvolvimento do projeto, sendo assim, código enxuto e funcional com lógica pertinente ao banco convencional.
